Walidacja strony HTML to proces sprawdzania kodu HTML strony internetowej pod kątem zgodności z określonymi standardami, najczęściej wyznaczonymi przez organizację W3C (World Wide Web Consortium). Dzięki walidacji można upewnić się, że kod jest poprawny.


Walidator to narzędzie lub program, który służy do sprawdzania poprawności danych lub kodu według określonych standardów lub reguł. W kontekście tworzenia stron internetowych walidator najczęściej odnosi się do narzędzia sprawdzającego kod HTML, CSS lub innych języków internetowych, by zapewnić jego zgodność z ustalonymi normami.
